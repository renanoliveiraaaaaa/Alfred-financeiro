# Política de Privacidade e Termos de Uso — Alfred Financeiro

## Política de Privacidade

- **Coleta de Dados:** Apenas dados necessários para funcionamento do serviço são coletados (nome, e-mail, dados financeiros inseridos pelo usuário).
- **Uso dos Dados:** Os dados são utilizados exclusivamente para prover a experiência do Alfred Financeiro. Não vendemos ou compartilhamos dados com terceiros.
- **Armazenamento:** Todos os dados são armazenados de forma segura no Supabase (PostgreSQL), com políticas de segurança (RLS) ativas.
- **Acesso:** Apenas o próprio usuário pode acessar seus dados. Administradores não têm acesso ao conteúdo financeiro individual.
- **Exclusão:** O usuário pode solicitar a exclusão total de sua conta e dados a qualquer momento (ver seção abaixo).
- **Cookies:** Utilizamos cookies apenas para autenticação e preferências de tema.
- **Contato:** Dúvidas sobre privacidade podem ser enviadas para o e-mail do suporte informado no rodapé do app.

## Termos de Uso

- **Aceite:** Ao criar uma conta, o usuário concorda com estes termos.
- **Uso Permitido:** O sistema é destinado a uso pessoal. É proibido uso para fins ilícitos ou automação maliciosa.
- **Responsabilidade:** O Alfred Financeiro não se responsabiliza por decisões financeiras tomadas com base nas informações do sistema.
- **Disponibilidade:** O serviço pode passar por manutenções e não garante uptime contínuo.
- **Alterações:** Esta política pode ser atualizada. Mudanças relevantes serão comunicadas no app.

## Exclusão de Conta

- O usuário pode solicitar a exclusão de sua conta e todos os dados associados pelo canal de suporte ou diretamente no app (quando disponível).
- Após a exclusão, os dados são removidos de forma irreversível em até 7 dias.
